1. 分类任务评估指标

在分类任务中，我们预测的目标是离散的类别。评估指标大多基于一个叫做**混淆矩阵（Confusion Matrix）**的表格。

混淆矩阵

对于一个二分类问题（预测结果为正例或负例），混淆矩阵如下：

预测为正例预测为负例
实际为正例TPFN
实际为负例FPTN
  • TP (True Positive): 真正例，模型预测为正，实际也为正。
  • TN (True Negative): 真负例，模型预测为负，实际也为负。
  • FP (False Positive): 假正例（第一类错误），模型预测为正，实际为负。
  • FN (False Negative): 假负例（第二类错误），模型预测为负，实际为正。

基于混淆矩阵，我们可以定义一系列核心指标：

准确率 (Accuracy)

  • 定义: 模型预测正确的样本数占总样本数的比例。
  • 公式: Accuracy = (TP + TN) / (TP + TN + FP + FN)
  • 优点: 非常直观，容易理解。
  • 缺点: 在样本不均衡的情况下，准确率是一个具有欺骗性的指标。例如，在一个99%的样本都是负例的数据集中，一个把所有样本都预测为负例的模型，其准确率高达99%，但这个模型毫无用处。

精确率 (Precision)

  • 定义: 在所有被模型预测为正例的样本中，实际也为正例的比例。也称为“查准率”。
  • 公式: Precision = TP / (TP + FP)
  • 解读: 回答了“我们预测的正例中，有多少是真正确的？”。它衡量了模型的预测纯度。
  • 应用场景: 当我们非常关心“假正例”带来的成本时，精确率很重要。例如，在垃圾邮件检测中，将一封重要邮件错误地标记为垃圾邮件（FP）的后果很严重，我们希望模型的精确率尽可能高。

召回率 (Recall)

  • 定义: 在所有实际为正例的样本中，被模型成功预测出来的比例。也称为“查全率”或“真正例率”（True Positive Rate）。
  • 公式: Recall = TP / (TP + FN)
  • 解读: 回答了“所有真实的正例中，我们找到了多少？”。它衡量了模型的覆盖能力。
  • 应用场景: 当我们非常关心“假负例”带来的成本时，召回率很重要。例如，在传染病筛查中，漏掉一个真正的病人（FN）的后果非常严重，我们希望模型的召回率尽可能高。

F1分数 (F1-Score)

  • 定义: 精确率和召回率的调和平均数。
  • 公式: F1 = 2 * (Precision * Recall) / (Precision + Recall)
  • 解读: F1分数是精确率和召回率的一个综合指标。只有当两者都较高时，F1分数才会高。它在样本不均衡的情况下，比准确率更能反映模型的真实性能。

精确率 vs. 召回率的权衡: 通常情况下，精确率和召回率是相互制约的。提高一方往往会导致另一方的降低。F1分数就是为了找到一个平衡点。

ROC曲线与AUC值

  • ROC曲线 (Receiver Operating Characteristic Curve): 这是一条以**假正例率（FPR = FP / (FP + TN)）**为横坐标，**真正例率（TPR，即召回率）**为纵坐标的曲线。曲线上的每个点对应于在不同分类阈值下模型的一对(FPR, TPR)值。
  • AUC (Area Under the Curve): ROC曲线下的面积。AUC的值介于0和1之间。
    • AUC = 1: 完美分类器。
    • AUC = 0.5: 随机猜测。
    • AUC < 0.5: 比随机猜测还差。
  • 优点: AUC是一个不依赖于特定分类阈值的评估指标，它衡量了模型在所有可能阈值下的总体排序能力。即使在样本极不均衡的情况下，AUC依然是一个稳定且可靠的指标。

2. 回归任务评估指标

在回归任务中，我们预测的目标是连续的数值。评估指标衡量的是预测值与真实值之间的“距离”或“误差”。

假设真实值为 y，预测值为 ŷ。

平均绝对误差 (Mean Absolute Error, MAE)

  • 公式: MAE = (1/n) * Σ|y_i - ŷ_i|
  • 解读: 计算了每个样本的预测误差的绝对值的平均数。它直观地反映了预测值的平均误差大小，并且单位与原始数据相同。

均方误差 (Mean Squared Error, MSE)

  • 公式: MSE = (1/n) * Σ(y_i - ŷ_i)^2
  • 解读: 计算了每个样本的预测误差的平方的平均数。与MAE相比，MSE对较大的误差（离群点）给予了更高的惩罚。但其单位是原始数据单位的平方，解释性稍差。

均方根误差 (Root Mean Squared Error, RMSE)

  • 公式: RMSE = sqrt(MSE)
  • 解读: 它是MSE的平方根。RMSE的优点在于其单位与原始数据相同，因此比MSE更易于解释。它保留了MSE对大误差更敏感的特性。

MAE vs. RMSE: 如果你认为所有误差的权重都是相同的，MAE更适合。如果你认为大误差比小误差更不可容忍，RMSE是更好的选择。

总结

没有一个评估指标是万能的。选择哪个指标，取决于你的具体业务场景和目标。

  • 对于分类问题，当样本均衡时，可以看准确率；当样本不均衡时，应优先关注F1分数和AUC。
  • 如果你更关心假正例（FP），关注精确率。
  • 如果你更关心假负例（FN），关注召回率。
  • 对于回归问题，MAE提供了对平均误差的直接衡量，而RMSE则放大了大误差的影响。

在实际项目中，通常需要综合考察多个指标，才能全面、客观地评估一个模型的优劣。
