# 打赏二维码

请将支付宝（及可选微信）收款码图片放在此目录，命名规则：

- **支付宝**：`alipay-2.png`、`alipay-5.png`、`alipay-10.png`、`alipay-50.png`、`alipay-100.png`、`alipay-1.png`（任意金额）
- **微信**（当 `reward_alipay_only` 为 false 时）：`wechat-2.png`、`wechat-5.png` 等

若只用一张支付宝码，可复制同一张图为以上文件名，或仅在 `hugo.toml` 中开启 `reward_alipay_only` 并放置 `alipay-2.png` 等至少一张。
