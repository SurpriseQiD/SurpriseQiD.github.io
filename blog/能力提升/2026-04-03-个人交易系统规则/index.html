    个人炒股交易系统规则（2026 版）

    本文用于个人交易系统建设与复盘，不构成任何投资建议。

    一、系统目标与底层原则

    1) 总目标

    • 第一目标：稳定控制回撤。
    • 第二目标：在可控风险下获取趋势收益。
    • 第三目标：把临场情绪决策，转换为规则决策。

    2) 底层原则

    • 先定义风险，再追求收益。
    • 先看市场环境，再看个股机会。
    • 先做确认，再做加仓。
    • 先执行系统，再复盘优化。

    二、市场环境过滤器（先判断能不能打）

    每个交易日先判断环境，再决定进攻或防守。

    1) 指数与广度

    • 若全市场普跌、下跌家数极高（如 4000 家以上），默认防守日。
    • 若指数缩量下行且主线普跌，默认“兑现优先”，不做激进追高。

    2) 节假日与事件风险

    • 节前 1-2 天：默认降低隔夜风险，以持币过节为优先策略。
    • 重大事件窗口（地缘、政策、业绩密集披露）：仓位自动下调一级。

    3) 板块状态

    • 主线一致性强：允许做趋势跟随。
    • 主线分歧加剧：降低交易频率，只做熟悉票与确认买点。
    • 无量拉升题材：不参与追单。

    三、标的池与优先级

    1) 标的池分层

    • A 层（核心熟悉票）：国晟科技、杭电股份、中利集团。
    • B 层（主线龙头候选）：医药、电力、半导体等板块中的高辨识度标的。
    • C 层（观察池）：消息驱动但尚未确认持续性的标的。

    2) 交易优先级

    • 优先做 A 层熟悉票的低风险节奏。
    • 其次做 B 层确认后的龙头跟随。
    • C 层只观察，不作为主要收益来源。

    四、仓位系统（账户级风控核心）

    1) 总仓位档位

    • 进攻档（市场强一致）：60%-80%。
    • 中性档（震荡分歧）：30%-50%。
    • 防守档（普跌/节前/高不确定）：0%-30%。

    2) 单笔风险预算

    • 单笔最大亏损预算：账户净值的 0.5%-1.0%。
    • 日内最大回撤阈值：账户净值的 2.0%。触发后当日停止主动进攻。
    • 连续两笔不及预期：立刻降频，仓位降到防守档。

    3) 仓位分批

    • 试错仓：30%。
    • 确认仓：30%。
    • 趋势延续仓：20%-40%（仅在原假设被新证据强化时启用）。
    • 禁止首笔重仓、禁止亏损摊平式加仓。

    五、入场系统（什么时候买）

    1) 必须同时满足的入场条件

    • 市场条件：不处于系统定义的强防守时段。
    • 板块条件：板块至少不是明显退潮。
    • 个股条件：趋势未破坏，且出现“低点右侧确认”信号。
    • 量价条件：不是无量脉冲，不在急拉末端追单。

    2) 你的专属入场纪律

    • 进货只做“跌下来后的右侧”，不做左侧赌反转。
    • 原计划触发价不到，不下单。
    • 若提前下单，默认视为纪律违例，仓位自动减半。

    3) 放弃交易规则

    • 买点不舒服，宁可空仓。
    • 直接拉板不给低风险位置，放弃追高。
    • 看不懂的票不做。

    六、出场系统（什么时候卖）

    1) 止损规则（先活下来）

    • 结构止损：跌破入场逻辑关键位，先减仓再清仓。
    • 价格止损：单笔达到预设损失线（如 -3% 到 -5%，按波动级别定义）必须执行。
    • 禁止“等一等就回来”的主观拖延。

    2) 止盈规则（锁利润）

    • 达到阶段目标收益先分批落袋。
    • 浮盈票优先兑现，不把利润暴露在高不确定隔夜风险中。
    • 高开过高（如 +6% 及以上）但承接转弱，优先兑现。

    3) 你的峰值出货规则（核心）

    • 一般冲高到 +5% 左右，已满足阶段止盈要求，不能贪。
    • 第一波快速拉升后，开始等待结构确认。
    • 若第二峰低于第一峰：第二峰后右侧立即出货。
    • 若第二峰高于第一峰但不能封板：在“低于最高峰后的首个峰值右侧”出货。
    • 出货必须“拉起来卖”，不在恐慌杀跌点乱砍。

    七、节奏与执行系统（盘前-盘中-盘后）

    1) 盘前（9:15 前完成）

    • 写三条线：入场触发线、失效止损线、减仓兑现线。
    • 给每个持仓写一个“若 A 则 B”预案，不允许盘中现编。
    • 确认当日仓位档位（进攻/中性/防守）。

    2) 盘中（只执行，不改逻辑）

    • 每次下单前必须回答：
      • 我现在是在执行计划，还是在情绪反应？
      • 这笔单的止损在哪里？
      • 若错了，我准备怎么退？
    • 若连续两次操作不及预期：当日进入保护模式（降仓、降频、少操作）。

    3) 盘后（系统升级时段）

    • 复盘每一笔交易的“计划一致性”。
    • 统计纪律违例次数，而不是只看盈亏。
    • 把当天错误转成下一交易日的参数修正。

    八、量化评估体系（周度必做）

    每周统计以下指标，系统才会进化：

    • 胜率（Win Rate）
    • 盈亏比（Payoff Ratio）
    • 最大回撤（Max Drawdown）
    • 平均持仓时长
    • 纪律违例次数（Discipline Violations）
    • 计划内交易占比（Plan Adherence）

    参考阈值

    • 若纪律违例次数 > 3 次/周：下一周仓位上限下调一级。
    • 若连续两周盈亏比 < 1：暂停激进策略，只做低风险确认单。
    • 若最大回撤超阈值：强制进入防守模式一周。

    九、你当前最关键的系统补丁

    基于近期实盘，当前最需要修复的不是选股能力，而是执行一致性。

    • 补丁 1：把“计划前置”做成硬规则，盘中不再改剧本。
    • 补丁 2：盈利单优先兑现，尤其节前和高分歧日。
    • 补丁 3：消灭“看好情绪覆盖风险规则”的行为。
    • 补丁 4：所有买卖点必须右侧确认，避免急拉追高与左侧抄底。

    十、每日交易检查清单（可直接复制）

    开盘前

    • 今天是进攻日、中性日，还是防守日？
    • 我今天总仓位上限是多少？
    • 每个持仓的止损线和兑现线是否写清？
    • 今天哪些情况出现后必须空仓观望？

    盘中

    • 本次下单是否满足预设触发条件？
    • 是否在低点右侧买、高点右侧卖？
    • 当前是否出现连续两笔不及预期，需进入保护模式？

    收盘后

    • 今日最大错误是什么？是否写成新规则？
    • 今日是否有纪律违例？原因是什么？
    • 明天要删除的坏动作是什么？

    十一、外部经验与本文融合依据

    本文规则除实盘经验外，还融合了外部常见风险管理原则：

    • 仓位管理：单笔风险预算与仓位按风险反推。
    • 止损机制：止损单能增强纪律，但需认识跳空与滑点风险。
    • 风险收益比：每笔交易应在入场前完成潜在损失与潜在收益评估，并动态调整。

    这些外部原则最终都服务同一件事：把主观判断放进客观流程，让系统长期可复用、可迭代。
